a "form"al rundown

  • "form" lets users input info & send!
  • "action" determines where the info goes!
  • "method" determines how the info is sent & processed
  • "input" adds field & "type" sets the attribute, which are:
  1. text = words/letters
  2. password = censors text input
  3. number = self-explanatory
  4. range = slider
  5. radio = radio button / single choice
  6. checkbox = checkbox / multiple choice
  7. submit = button
  8. text + "list" attribute -> "datalist" element if list of input & id of datalist are same
  9. + "required" = field needs to be answered
  • "select" + "option" = dropdown list
  • "datalist" + "option" = search choices
  • "textarea" = customizable text input field
  • and submitting form = "name" of fields with input & "value" of them are sent as name=value pairs
